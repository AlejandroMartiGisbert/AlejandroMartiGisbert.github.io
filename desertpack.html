Desert Pack.

A project combining blender and unity.

Project Overview.

3D models in blender and scene in Unity

This package contains a variety of assets ready to use for your game to create a desert environment.

You can read more about this asset in my itch-io page.

Content of the package

  • 8 Demo Scenes
  • 1 Rigged Character
  • 18 Buildings with 2 prefabs(with and without Wood Parts)
  • 43 Props
  • 2 Hexagon Tiles + Water
  • 2 Trees + 3 Cactus
  • 4 Rocks
  • 2 Particle Systems (Rain + Fire)
  • 1 Post Processing Profile

The file includes the Unity Project with the lighting setup and a unitypackage with only the files of the asset pack.
